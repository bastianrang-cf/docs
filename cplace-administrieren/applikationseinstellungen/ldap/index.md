# LDAP

Parameters of the LDAP interface are displayed in the **LDAP** tab,
provided that an LDAP interface has been configured. It is only possible
to configure the LDAP interface by way of server-based code
modifications.

If you would like to integrate LDAP, contact your collaboration Factory
contact person.
