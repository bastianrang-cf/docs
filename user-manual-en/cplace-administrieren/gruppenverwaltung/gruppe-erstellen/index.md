# Creating a group

To create a group, proceed as follows:

::: {.note .note-prerequisite .with-title}
**Prerequisite**

You require the global "Create groups" permission.
:::

1.  Open the "User Menu".

2.  Open the **All Users & Groups** menu option.

    []{.fas .fa-check-square} The "Users and Groups" page is displayed.

3.  Click the **Groups** tab.

    []{.fas .fa-check-square} The **Groups** tab is opened, and a table
    with all the groups is displayed.

4.  Click the **\[New Group\]** button.

    []{.fas .fa-check-square} The "Create new group" dialog is
    displayed.

5.  Enter a name for the group.

6.  If necessary, configure the other settings of the group. A
    description of the possible settings can be found under ["Editing
    group
    settings".](/user-manual-en/cplace-administrieren/gruppenverwaltung/gruppeneinstellungen_bearbeiten/)

7.  Click the **\[Save\]** button to create the new group.

    []{.fas .fa-check-square} The new group is created and the group
    page is opened.
